Simple definitions for terms used in this guide.

## Core Terms

**VPS (Virtual Private Server)**  
A remote computer you rent. Runs 24/7 in a data center.

**SSH (Secure Shell)**  
The encrypted connection method to access your VPS remotely.

**Root**  
The admin/superuser account on Linux. Has full control.

**IP Address**  
Your VPS's address on the internet (like `123.45.67.89`).

**Terminal**  
The black window where you type commands.

**Port**  
A "door number" for connections. SSH uses port 22.

## Connection Tools

**Bitvise**  
Windows SSH client with terminal + file transfer.

**VS Code (Visual Studio Code)**  
Code editor that can connect to VPS remotely.

**PuTTY**  
Simple, lightweight SSH terminal for Windows.

## Linux Terms

**Directory**  
A folder. Linux uses `/` instead of `\`.

**Command**  
An instruction you type in the terminal.

**sudo**  
"SuperUser DO" - run a command with admin privileges.

**apt**  
Package manager for Ubuntu/Debian. Installs software.

**pwd**  
Print Working Directory - shows where you are.

**ls**  
List - shows files in current directory.

**cd**  
Change Directory - move to different folder.

## Security Terms

**Firewall**  
Blocks unwanted network connections.

**UFW (Uncomplicated Firewall)**  
Easy firewall tool for Ubuntu.

**Fail2Ban**  
Automatically blocks IPs that attack your server.

**SSH Keys**  
Pair of cryptographic keys (more secure than passwords).

**Authentication**  
Proving who you are (via password or SSH key).

## Node Operator Terms

**Node**  
A computer participating in a blockchain network.

**Blockchain**  
Distributed ledger/database across many computers.

**Data Center**  
Building full of servers with fast internet and backup power.

**Uptime**  
How long a server stays online (99.9% = very reliable).

**Bandwidth**  
Amount of data that can be transferred per month.

## File System

**/ (Root Directory)**  
Top level of file system (not to confuse with root user).

**/root**  
Home folder for the root user.

**/home**  
Contains home folders for regular users.

**/var/log**  
Log files (system records what happened).

**/etc**  
Configuration files for system and software.

## Common Acronyms

**CLI** - Command Line Interface (the terminal)  
**GUI** - Graphical User Interface (windows and buttons)  
**RAM** - Random Access Memory (temporary memory)  
**CPU** - Central Processing Unit (processor)  
**SSD** - Solid State Drive (fast storage)  
**OS** - Operating System (Linux, Windows, etc.)
